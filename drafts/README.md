## Promotion flow (when you do)

1. Read the draft.
2. For lineage / counterargument: decide format (essay section, dedicated page, appendix), rewrite in voice, ship.
3. Delete each draft after promotion, or leave for future reference — the drafts are provenance, not pre-canon.

## Why this folder still exists

Provenance. The shipped propositions in `src/content/propositions.ts` show the *result*; these drafts show the *case* that produced them. Useful when revisiting an objection or auditing why a phrasing was chosen.
